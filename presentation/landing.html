Try it now

Wire it into a build in about five lines.

# .github/workflows/ci.yml — after your build
- uses: alexmond/unitrack/action@v0
  with:
    url: https://unitrack.example.com
    token: ${{ secrets.UNITRACK_TOKEN }}
    # junit + jacoco globs auto-detected
# upload this build's reports (metadata auto-detected)
unitrack upload --project checkout \
  --junit '**/surefire-reports/*.xml' \
  --jacoco '**/jacoco.xml'

# then gate the build
unitrack gate --project checkout --min-coverage 80
# self-host in one command (H2 demo mode)
docker run --rm -p 8080:8080 \
  ghcr.io/alexmond/unitrack:0.4.0

# → open http://localhost:8080 · Postgres compose for prod
